## ✨ Overview

**Velvoria Web** is the official web client of the Velvoria marketplace. It is built as a next-generation luxury e-commerce platform featuring premium aesthetics, live shopping streams, and responsive visual design. Designed with minimalist layout, elegant color schemes (Deep Navy & Rose Gold), and smooth transitions to provide a first-class shopping experience for high-end consumers.

---

## 🚀 Key Features

* **💎 Luxury & Minimalist UI** - Modern interface using Playfair Display & Inter typography.
* **🎥 Live Shopping Integration** - Immersive video streams directly embedded into the marketplace.
* **📦 Multi-Vendor Scaffolding** - Separate UI views for buyers and premium vendors.
* **⚡ Blazing Fast Performance** - Powered by Vite and React TS with optimized code splitting.
* **🌀 Rich Micro-Interactions** - Animated transitions powered by Framer Motion.

---

## 🛠️ Tech Stack

- **Framework:** React 18+ (TypeScript)
- **Bundler:** Vite
- **Styling:** Tailwind CSS (with customizable system parameters)
- **Icons:** Lucide React
- **Animation:** Framer Motion

## 📥 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Naufall18/velvoria-web.git
   cd velvoria-web
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser to see the live app.

4. **Build for Production**
   ```bash
   npm run build
   ```
